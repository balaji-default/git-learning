git clone command

git clone command is used for cloning a repository from a remote server to the local machine.


git status command

status are of many types
  • untracked
  • modified
  • staged
  • unmodified

      • modified (M)- changed the content in the file
      untracked (U) - files are yet to be added to git ( not tracked by git )
      staged(A) - files are added to the staging area ( ready to be committed )
      commit - files are added and commited to the repository -unchanaged (unmodified)
